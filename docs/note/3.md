# 创建仓库

用vscode打开写代码的文件夹

![public](/public3/2025_12_04_20_36_21.jpg)

在docs文件夹里面1创建一个专门存放笔记的文件夹，图中专门存放把笔记的文件夹叫note，然后在其中创建笔记1、2、3.......（以.md为后缀命名文件）

在markdown中创建同名文件进行笔记记录，记录完成后将笔记转化为源码模式，复制代码

![public](/public3/2025_12_06_10_40_59.jpg)

回到vscode将复制的代码粘贴到你创建的笔记文件里

![public](/public3/2025_12_06_10_52_50.jpg)

打开config.mts，找到items，添加你的笔记名称和链接（图中以笔记1.md为例）

![public](/public3/2025_12_06_10_55_58.jpg)

## 打包代码

打开终端，在终端输入命令行：npm run docs:build

![public](/public3/2025_12_06_11_12_01.jpg)

如图所示打包成功。

打开文件管理器，找到自己写代码的文件夹，根据下图所示位置找到dist文件，将里面的文件和文件夹复制（这一步就是复制刚刚打包好的东西）

![public](/public3/2025_12_06_11_26_53.jpg)

打开自己的本地仓库（图中的本地仓库为blog），将刚刚复制的东西全部粘贴到自己的本地仓库

![public](/public3/2025_12_06_11_30_30.jpg)

粘贴完成后，用vscode打开自己的本地仓库，也就是你刚刚粘贴的地方，开始上传自己的笔记

新建终端，打入以下指令：

git add .

![public](/public3/2025_12_07_14_48_06.jpg)

git commit -m "备注"

![public](/public3/2025_12_07_14_48_19.jpg)

git pull

![public](/public3/2025_12_07_14_48_31.jpg)

git push

![public](/public3/2025_12_07_14_49_06.jpg)

如图所示打开自己github的仓库

![public](/public3/2025_12_07_14_54_19.jpg)

打开网站以后就可以看见自己的笔记了

![public](/public3/2025_12_07_14_55_22.jpg)
